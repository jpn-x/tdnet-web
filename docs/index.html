📖 使い方ガイド

■ 初回セットアップ（必須）

▸ Cloudflare Worker URL の設定

このWebツールは Cloudflare Worker 経由でTDnetのデータを取得します。

1. Cloudflare アカウントを作成（無料）

2. Worker をデプロイ（README.md の手順参照）

3. [設定] → Worker URL に Worker の URL を貼り付けて保存

■ 自動更新

▸ 更新間隔プルダウン

ヘッダーの [1秒〜60分] で更新間隔を選択できます。

▸ 15分刻み強制更新

毎時 0・15・30・45 分の1秒後に自動でデータを再取得します。

▸ 手動更新

[↻ 手動更新する] をクリックすると即時取得します。

■ 日付ナビゲーション

▸ ◀ / ▶ ボタン

前日・翌日のデータに切り替えます。

▸ 日付クリック

📅 日付部分をクリックするとカレンダーが開きます。

▸ [今日に戻る]

過去日付表示中のみ表示。本日のリアルタイムデータに戻ります。

■ ページ取得モード

▸ [全ページ取得]（青）

その日の全開示を漏れなく取得します。ページ数はステータスに表示。

▸ [1ページのみ]（グレー）

最初の100件のみ取得。速度優先の場合に使用。

■ データ表示の見方

▸ 行の見方

● オレンジドット = 新着（前回更新以降に追加された開示）

緑色文字 = 既に開いた（閲覧済み）開示

白/黒文字 = 未閲覧の開示

※ 閲覧済み情報はブラウザのlocalStorageに保存されます

■ REIT・ETF フィルター

[設定] → 「REIT・ETFの開示を除外する」チェックで切替。

J-REIT・ETF・ETNの定型開示を除外し個別株のみ表示します。
